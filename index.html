PolitiKweli: A Swahili-English Code-Switched Classification Dataset

The evolution of languages has resulted in a new generation of speakers using simplified forms of languages or mixing two or more languages in a single utterance. The code-switched Swahili-English dataset built from data collected from Kenyan users of the social media platform Twitter (now X) can be used for various machine learning tasks. The dataset was originally curated for text classification.
